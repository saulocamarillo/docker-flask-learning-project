# Simple Flask Application

This repository contains a minimal Flask application used for learning Docker, inspired by the example shown in Mumshad Mannambeth's Docker course.

---


Running the App Locally

1. Update the system

   apt-get update


2. Install Python3 venv

   apt-get install -y python3.14-venv


4- Select the source

   source venv/bin/activate


5. Install Pip (extencion de python)

   apt-get install pyton3-pip


6. Install the flask dependency

   pip install flask


7. Run the application:

   python app.py


8. Open your browser at:

   http://localhost:5000
